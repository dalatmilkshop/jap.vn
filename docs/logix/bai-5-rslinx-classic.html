In RSLogix 5000 programming, RSLinx Classic is used for Download, Upload, Online, etc.

Usage steps:

  1. Connect your PC to the target Controllers (e.g., via Ethernet Switch, same IP subnet), set PC network card IP (e.g., 192.168.1.190).
  2. Start RSLinx Classic
  4. Select Communication > Driver
  5. Select Ethernet/IP and Add Net
  8. OK, select the network card connected to the Controller
  10. Add New and see Status is Running as below
  12. Return to main screen, expand the new Driver, connected Controllers will appear as below.
Notes:
  • For old AB PLCs (PLC-5, SLC-500, etc.), select the appropriate Driver (DH+, DF1, ...). New Controllers use Ethernet, much easier.
  • You can view IO modules on each Chassis directly in RSLinx.
  • IP addresses of network devices can be changed right in RSLinx Classic.
  • RSLinx Classic can act as an OPC Server for other applications.
