If the Firmware and Revision of Rslogix 5000 do not match, you must upgrade the Controller's Firmware or install the matching Rslogix 5000 version. New Controllers also require Firmware upgrade as they ship with factory default (usually 1.008).

Upgrade Firmware steps:

  1. When installing Rslogix 5000, the FLASH Programming Tool is also installed in Program Files. This is used to Upgrade/Downgrade Firmware for AB controllers.
  2. Preparation for Upgrade/Downgrade:
    • Backup your program, as upgrading/downgrading will erase the CPU program
    • Connect PC to the target Controller
    • Switch the CPU key to PROGRAM mode
  4. Run Flash Programming Tool, NEXT
  6. Select the Controller or Module to upgrade. Here, select Controllogix L75 CPU
  7. NEXT
  9. OK
  11. Select the version to Upgrade or Downgrade
  12. NEXT
  14. Finish
  16. Confirm
  17. YES
  19. Upgrade in progress
  21. Upgrade complete notification appears. If error, a red error message will show
Notes:
  • Upgrading/Downgrading Firmware is normal, but many feel anxious as it's rarely done.
  • Keep CPU powered and PC-CPU connection stable during upgrade. If interrupted, CPU reverts to factory firmware and must be upgraded again. This can sometimes cause errors or damage.
  • Use Ethernet or USB (for new CPUs), avoid RS-232 as it's slow and unreliable.
  • Upgrade time: 3-5 minutes (via Ethernet)
