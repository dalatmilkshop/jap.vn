Hardware in this lesson (and next):

  • 10 Slot Chassis
  • Controllogix L75 CPU in Slot 0
  • 32-point DC Input Module in Slot 5
  • 32-point DC Output Module in Slot 6

Create a new project with RSLogix 5000 as follows:

From main screen, select New or File > New

In this window, select correct CPU parameters:

  • CPU type (here: 1756-L75)
  • Revision: select RSLogix 5000 version (here: Version 20). Must match hardware firmware (see lesson 6: Firmware)
  • Name: Project name (required, shown in RSLinx next to CPU)
  • Chassis Type: select correct chassis (here: 10 Slot)
  • Slot: select slot for CPU
  • Create In: select file save location

Click OK when done.

Project is created as below:

Add 2 IO Modules. To add, right-click IO Configuration and select New Module:

New window appears:

Type or select IB32, choose from list, then Create.

In new window, name the module and select correct slot (here: Slot 5), then OK. Another window appears, click OK (you'll learn to configure as needed later)

IB32 module now appears in I/O Configuration. Similarly, add OB32 module as below:

With Compactlogix

Creating a new project with Compactlogix is similar to Controllogix above, with some small differences:

To add new IO, select New Module at Expansion I/O Module.

Notes:
  • Above are basic steps to configure project hardware in RSLogix 5000.
  • For Controllogix (1756- modules), you can create Online using RSLogix 5000 Discovery
