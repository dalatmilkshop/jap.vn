Rockwell ControlLogix Standard Controllers

Gi√°p VƒÉn V·ªπ ‚Äì 2012
  • This is Rockwell Automation‚Äôs large PAC line, designed for complex control applications, DCS, multiple network types, and high I/O count (over 1,000).
  • There are two main product lines: 1765-L6x and 1756-L7x. The L7x series, introduced in recent years, offers new features and double the performance of the older 1756-L6x, but 1756-L6x is still common in factories.
  • Maximum CPU memory is 32MB (1756-L75)
  • Maximum I/O capacity: 128,000 I/O
  • SD card (L7x) for program backup
  • USB 2.0 port for programming (L7x)
  • Integrated Motion
  • No battery required.

Chassis, Slot, and Module

  • The chassis is a frame with a power supply and slots for modules. Chassis come in various sizes: 4, 7, 10, 13, and 17 slots.
  • The power supply provides power to all modules in the chassis. There are many types (AC, DC, Redundant) and capacities, and must be purchased separately (not included with chassis).
  • Slots are for inserting modules; all 1756-xxx modules can be installed in any slot.
  • Modules include CPU, I/O modules, and network communication modules. Modules can be installed in any slot.
    • Multiple CPUs can be installed in one chassis
    • Modules can be hot-swapped (removed/inserted without power-off)
    • Chassis can have no CPU (acting as a Remote IO station)

Applications

Controllogix is suitable for:

  • Complex control, integrated systems like DCS, Batch
  • Redundant (fault-tolerant) configurations
  • Large, distributed IO (up to 128,000 I/O)
  • Applications with multiple industrial network types: Controlnet, Ethernet, DeviceNet, DH+, etc.
